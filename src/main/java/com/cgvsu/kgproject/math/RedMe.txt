МАТРИЦА 4*4 (FLOAT)
Конструкторы:
Matrix4f(): Создает пустую матрицу 4x4.
Matrix4f(float m00, ... , float m33): Конструктор с параметрами для создания матрицы с заданными значениями.
Matrix4f(Matrix4f matrix4f): Конструктор копирования для создания новой матрицы на основе существующей.
Matrix4f(float[] vertices): Конструктор для создания матрицы из массива значений.
Matrix4f(float[][] matrix): Дополнительный конструктор для создания матрицы на основе двумерного массива.

Метод mul(Matrix4f m1): Умножает текущую матрицу на переданную матрицу m1 и сохраняет результат в текущей матрице.
Метод multiplyByVector3(Vector3f vertex): Умножает текущую матрицу на вектор Vector3f и возвращает новый вектор.
Статические методы для создания специальных матриц:

identityMatrix(): Возвращает единичную матрицу.
rotationMatrix(float angleX, float angleY, float angleZ): Возвращает матрицу поворота по осям X, Y и Z.
scaleMatrix(float scaleX, float scaleY, float scaleZ): Возвращает матрицу масштабирования.
translationMatrix(float translateX, float translateY, float translateZ): Возвращает матрицу трансляции. Используется для перемещения объекта в трехмерном пространстве
transpose(): Возвращает транспонированную матрицу.


ВЕКТОР 3 (FLOAT)
Конструкторы:
Vector3f(): Создает пустой вектор.
Vector3f(float x, float y, float z): Создает вектор с указанными координатами.
Vector3f(Vector3f vector3f): Конструктор копирования.

Метод sub(Vector3f to, Vector3f from): Вычисляет разницу между двумя векторами.
Метод cross(Vector3f v1, Vector3f v2): Вычисляет векторное произведение между двумя векторами (v1 x v2) и сохраняет результат в текущем векторе.
Метод normalize(): Нормализует вектор, делая его длину равной 1.

Метод dot(Vector3f v1): Вычисляет скалярное произведение между текущим вектором и переданным вектором.
Метод add(Vector3f t1, Vector3f t2): Складывает два вектора (t1 + t2) и сохраняет результат в текущем векторе.
Метод add(Vector3f t1): Складывает текущий вектор с другим вектором t1 и сохраняет результат в текущем векторе.
Методы scaleX(float scale), scaleY(float scale), scaleZ(float scale): Умножают соответствующую координату вектора на заданный множитель.
Статический метод fromTwoPoints(Vector3f vertex1, Vector3f vertex2): Создает вектор, представляющий разницу между двумя точками в пространстве.
Статический метод sum(List<Vector3f> vectors): Суммирует все векторы из списка и возвращает результат.
Метод length(): Возвращает длину вектора.
Метод hashCode(): Переопределенный метод для вычисления хэш-кода.
Метод divide(float num): Делит каждую координату вектора на заданное число num и возвращает новый вектор.
Метод equals(Object o): Переопределенный метод для сравнения векторов на равенство.
